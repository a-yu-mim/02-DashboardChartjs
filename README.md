# 📊 Dashboard

Simple dashboard developed with **HTML, CSS, and Chart.js**, displaying temperature and humidity charts.

---

## Features

- Line chart with:
  - Temperature (°C)  
  - Humidity (%)  

- Bar chart with:
  - Monthly average temperature  
  - Monthly average humidity
