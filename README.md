# 📊 Dashboard

Dashboard simples desenvolvida com **HTML, CSS e Chart.js**, que exibe gráficos de temperatura e umidade.

---

 Funcionalidades
- 📈 Gráfico de linha com:
  - Temperatura (°C)
  - Umidade (%)
- 📊 Gráfico de barras com:
  - Temperatura média mensal
  - Umidade média mensal
